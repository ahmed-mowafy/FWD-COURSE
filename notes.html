The Web

The Web (or World Wide Web) is a collection of documents and other files. Most of the documents are made using Hypertext Markup Language (HTML).

What is HTML?

HTML stands for Hypertext Markup Language. It's used to define the structure or layout of a web page. Notice that there's one main heading, followed by two sub-headings that are the same size.
